# 介绍各种容易记错的名词

## WebSocket 和 Socket套接字

前者是长连接，是基于HTTP和TCP的高层协议
后者是操作系统层面上的端点，更底层，是程序之间通信的基础

## ajax，fetch 和 axios

> 不知道为什么，老是记错三者，可能是老用封装好的东西的缘故（🥀

首先先后顺序上，ajax（2005）晚于fetch（现代标准，2015）和axios（强大功能流行，语法最简洁，2014）

实现上，Ajax用的XMLHttpRequest, Fetch API 是浏览器原生API，Axios浏览器端为XMLHttpRequest，Node.js端为Http
